Le chat noir court dans le jardin. Le chat aime chasser les souris. Le soleil brille sur la ville.
Les oiseaux chantent joyeusement près de la fenêtre. Il y a des chats partout dans ce quartier.
L'homme regarde le chat sur le toit. Les souris fuient quand elles voient le chat.
Les chiens aboient après le chat, mais celui-ci est trop rapide. Ce chat est vraiment malin.
Les chats aiment se reposer au soleil pendant l'après-midi. Les oiseaux continuent de chanter.
